# 🛒 Ecom Market - Static E-commerce Template

A modern, responsive e-commerce website template built with HTML, CSS, and JavaScript. This is a static frontend template ready to be integrated with PHP and MySQL backend.

## 📋 Project Overview

Ecom Market is a complete frontend template for an e-commerce website. It features a modern design, responsive layout, and interactive components. This template is designed to be easily integrated with a custom PHP/MySQL backend.

## 🔗 Repository

**GitHub Repository:** [https://github.com/mabbasdev/ecom-market](https://github.com/mabbasdev/ecom-market)

## ✨ Features

### Pages Included
- 🏠 **Homepage** - Hero section, featured products, categories
- 🛍️ **Shop/Products Page** - Product listing with grid/list view
- 📄 **Product Details Page** - Individual product view with image gallery
- 🛒 **Shopping Cart Page** - Cart management interface
- 💳 **Checkout Page** - Order form and summary
- 👤 **Login/Register Pages** - Authentication forms
- 📞 **Contact Page** - Contact form and information
- ℹ️ **About Page** - Company information
- 🔍 **Search Results Page** - Product search interface
- and much more coming soon!

### Components
- Responsive navigation bar (mobile-friendly)
- Hero banner section
- Product cards with hover effects
- Category showcase
- Testimonials section
- Newsletter subscription form
- Footer with links and social media
- Breadcrumb navigation
- Pagination component
- Modal dialogs (quick view, etc.)
- Back to top button

### Interactive Features (JavaScript)
- Product image gallery with zoom
- Quantity selector
- Add to cart animations
- Price calculator
- Mobile menu toggle
- Smooth scrolling
- Form validation
- Search suggestions

## 🛠️ Technologies Used

### Frontend
- **HTML5** - Semantic markup
- **CSS3** - Custom styling with Flexbox and Grid
- **JavaScript (ES6+)** - Interactive functionality
- **Font Awesome** - Icons
- **Google Fonts** - Typography

### No Backend Yet
This is currently a **static template**. The plan is to implement:
- **PHP** - Server-side logic
- **MySQL** - Database management
